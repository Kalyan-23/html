OPERATING SYSTEM

An Operating System (OS) is a system software that manages computer hardware, software, and resources, and provides common services for computer programs. It acts as an intermediary between users and the computer hardware. The OS facilitates communication between the hardware and application software, enabling tasks like executing programs, file management, and networking.


Types of Operating Systems

Operating systems are categorized based on their functionality, purpose, and the types of devices they support. Here are the primary types:


  1. Batch Operating System
  • Processes similar jobs in batches without user interaction.
    Suitable for environments like payroll processing or billing systems.
    Examples: IBM OS/360, early versions of UNIX.
